Description

uTrade is an E-commerce web application were users can create account, sell, buy and trade products with other users.
Users will be able to post products to sell or trade, where other users will have the option to buy or trade the posted product

Priority

  • Create account
  • Post item
  • Buy item
  • Trade item

Non-Priority

  • Chat with sellers
  • Account validation
  • Interactive Graphics

Languages and Platform

  • Node.js
  • Express framework
  • Javascript
  • HTML/CSS
  • Azure SQL

Technical features learned in class

full utilization of HTML/CSS/Javascript/Node.js to develop fully functional web application.

Technical features that goes beyond

Fully dynamic web application that will communicate with database to grab and post elements.
